# 📑 Laboratórios de Usabilidade - UNIFACS

Este repositório centraliza o desenvolvimento de atividades práticas e resoluções de questões da disciplina de **Usabilidade** do curso na UNIFACS. O foco principal é a aplicação de conceitos de experiência do usuário (UX) e design de interface (UI) através do desenvolvimento web moderno.

## 🎯 Objetivos do Projeto

O objetivo destes laboratórios é consolidar conhecimentos sobre como criar interfaces que não sejam apenas funcionais, mas também intuitivas, acessíveis e eficientes para o usuário final.

Os principais tópicos explorados incluem:

* **Heurísticas de Nielsen:** Aplicação prática das 10 regras de ouro da usabilidade.
* **Acessibilidade (A11y):** Desenvolvimento de componentes inclusivos.
* **Design Responsivo:** Interfaces que se adaptam a diferentes dispositivos e resoluções.
* **Arquitetura de Informação:** Organização lógica de elementos e fluxo de navegação.

## 🛠️ Tecnologias Utilizadas

O ecossistema de desenvolvimento foca em ferramentas amplamente utilizadas no mercado:

* **HTML5 & CSS3:** Estruturação semântica e estilização avançada (Flexbox/Grid).
* **JavaScript (ES6+):** Lógica de interação e manipulação dinâmica de dados.
* **React:** Construção de interfaces baseadas em componentes reutilizáveis.

## 🚀 O que este repositório demonstra para recrutadores

Este repositório vai além do código; ele reflete competências em:

1. **Componentização:** Criação de código modular e de fácil manutenção.
2. **Clean Code:** Preocupação com a leitura e organização lógica dos arquivos.
3. **Foco no Usuário:** Demonstra que o desenvolvedor compreende a importância da jornada do usuário, não apenas a implementação técnica.
4. **Versionamento:** Uso profissional do Git para controle de progresso acadêmico.
